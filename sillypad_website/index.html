Its a very, very silly mini keyboard

Make it yourself!

SillyPad is an open source, 3D printed mini keyboard

Open source

All code, models and PCBs are available on Github under GPLv3 for hacking and customisation.

Uses Via and QMK

Customise with the popular program VIA, or code your own firmware with QMK.

3D Printed

The case is 3D printed, making it easy to customize colors and styles as long as you have access to a 3D printer.

Compact and portable

Small and lightweight, SillyPad fits easily in your bag or pocket perfect for travel or any workspace.

But what is Sillypad really?

SillyPad is a macropad with 16 keys, a 0.96" 128x64 single colour OLED screen and 2 rotary encoders. All powered by a USB C Rasberry Pi Pico running QMK, compatible with VIA.
